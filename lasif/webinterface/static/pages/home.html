The LASIF webinterface is an experimental feature meant to facilitate data exploration and to visually see what's going on.

It will show the state of a LASIF project at the point in time you executed $ lasif serve. If you change data in the project after that point in time, the behaviour of the webinterface is not defined. Just restart the server in that case.

If you have any ideas what else you would like to see in the webinterface or if there are any problems, please let me know.
